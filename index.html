🎯 How It Works

  • Create Invoice: Fill in customer details, add line items, and click "Record to Accounting"
  • Create Purchase Order: Fill in supplier details, add line items, and click "Record to Accounting"
  • Track Payments: Invoices recorded as income, POs recorded as expenses
  • Monthly Reports: View income vs expenses for each month
  • Year-End Summary: Export complete P&L statement as Excel file
  • Auto Year Rotation: System auto-creates entries for new years

💾 Data Storage

  • All data stored in browser memory (localStorage)
  • Export to Excel for backup & record keeping
  • Data persists until browser cache is cleared
  • Keep exported Excel files as permanent records
